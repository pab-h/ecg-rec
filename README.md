# ECG-REC

Este repositório contém o código para treinamento e avaliação de um modelo de Deep Learning voltado à reconstrução de eletrocardiogramas (ECG).

## Descrição

O objetivo deste projeto é reconstruir traçados de ECG completos a partir de um subconjunto de derivações, utilizando técnicas de aprendizado profundo. O modelo pode ser aplicado para estudos clínicos, pesquisas em cardiologia e aprimoramento de datasets com derivações incompletas.
